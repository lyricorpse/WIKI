Analysis I (MATH 521)¶

Last update: 2016-02-26

Ordered set¶

Let \(S\) be a set. An order on \(S\) is a relation, denoted by \(<\), with the following two properties:

  • If \(x\in S\) and \(y\in S\) then one and only one of the statements below is true:

    \[\begin{split}x<y, \hspace{1em}x=y, \hspace{1em}y<x\end{split}\]

  • If \(x,y,z\in S\), if \(x<y\) and \(y<z\), then \(x<z\).

An ordered set is a set \(S\) in which an order is defined.

Bounds¶

Upper bound¶

Suppose \(S\) is an ordered set, and \(E\subset S\). If \(\exists \beta \in S\) such that \(x\le\beta~\forall x\in E\), we say that \(E\) is bounded above, and call \(\beta\) an upper bound of \(E\).

Lower bound¶

Suppose \(S\) is an ordered set, and \(E\subset S\). If \(\exists \beta \in S\) such that \(x\ge\beta~\forall x\in E\), we say that \(E\) is bounded below, and call \(\beta\) an lower bound of \(E\).

Supremum¶

Suppose \(S\) is an ordered set, \(E\subset S\) is bounded above. Suppose \(\exists \alpha\in S\) with the following two properties:

  • \(\alpha\) is an upper bound of \(E\).
  • If \(\gamma < \alpha\), then \(\gamma\) is not an upper bound of \(E\).

Then \(\alpha\) is called the least upper bound or supremum of \(E\), and is denoted as \(\alpha = \sup E\).

Infimum¶

Suppose \(S\) is an ordered set, \(E\subset S\) is bounded above. Suppose \(\exists \alpha\in S\) with the following two properties:

  • \(\alpha\) is an lower bound of \(E\).
  • If \(\gamma > \alpha\), then \(\gamma\) is not an lower bound of \(E\).

Then \(\alpha\) is called the greatest lower bound or infimum of \(E\), and is denoted as \(\alpha = \inf E\).

Least-upper-bound property¶

An ordered set \(S\) is said to have the least-upper-bound (LUB) property if the following is true: If \(E\subset S\), \(E\ne \emptyset\), and \(E\) is bounded above, then \(\exists \sup E \in S\).

Every ordered set with the LUB property also has the GLB property.

Greatest-lower-bound property¶

An ordered set \(S\) is said to have the greast-lower-bound (GLB) property if the following is true: If \(E\subset S\), \(E\ne \emptyset\), and \(E\) is bounded below, then \(\exists \inf E \in S\).

Every ordered set with the GLB property also has the LUB property.

Metric space¶

A set \(X\), whose elements we shall call points, is said to be a metric space if with any two points \(p\) and \(q\) of \(X\) there is associated a real number \(d(p,1)\), called the distance from \(p\) to \(q\), such that

  • \[\begin{split}d(p,q) \begin{cases} > 0, \text{ if }p\ne q, \\ = 0, \text{ if }p = q. \\ \end{cases}\end{split}\]

  • \(d(p,q) = d(q,p)\).

  • \(d(p,q) \le d(p,r) + d(r,q) ~\forall r\in X\).

Neighborhood¶

Let \(X\) be a metric space, \(p\) is a point in \(X\).

A neighborhood of \(p\) is a set \(N_r(p) = \{q\in X | d(p,q) < r\}\), for some radius \(r>0\).

Theorem:

  • Every neighborhood is an open set.

Limit point¶

Let \(X\) be a metric space, \(E\subset X\), a point \(p\in X\).

If every neighborhood of \(p\) contains \(q\ne p\) such that \(q\in E\), then \(p\) is a limit point of \(E\).

In other words, we say \(p\in E\) is a limit point of \(E\) if: \(\forall r>0, \exists q\in E, q\ne p\), such that \(d(p,q) < r\).

Or, we say \(p\in E\) is a limit point of \(E\) if: \(p\) is infinitely close to a point \(q\ne p\) such that \(q\in E\).

Isolated point¶

Let \(X\) be a metric space, \(E\subset X\), a point \(p\in X\).

If \(p\in E\) is not a limit point of \(E\), then \(p\) is called an isolated point of \(E\).

Interior point¶

Let \(X\) be a metric space, \(E\subset X\), a point \(p\in X\).

If \(\exists N_r(p)\) such that \(N_r(p) \subset E\), then \(p\) is an interior point of \(E\).

In other words, we say \(p\) is an interior point of \(E\) if we can put a neighborhood \(N\) (an open ball) centered at \(p\) such that \(N\subset E\).

Closed set¶

Let \(X\) be a metric space, \(E\subset X\).

We say \(E\) is a closed set if \(E\) contains its every limit point.

In other words, if a point \(p\) is the limit point of \(E\), then \(p\in E\).

Open set¶

Let \(X\) be a metric space, \(E\subset X\).

\(E\) is an open set if every point of \(E\) is an interior point of \(E\).

Complement set¶

Let \(X\) be a metric space, \(E\subset X\).

The complement set of \(E\) (denoted by \(E^c = X-E\)) is the set of all points \(p\in X\) such that \(p\notin E\).

Perfect set¶

Let \(X\) be a metric space, \(E\subset X\).

If \(E\) is a closed set (every limit point of \(E\) is a point of \(E\)) and if every point of \(E\) is a limit point of \(E\).

Bounded set¶

Let \(X\) be a metric space, \(E\subset X\).

\(E\) is a bounded set if \(\exists M\in\mathbb{R}\) and a point \(q\in X\) such that \(d(p,q)<M~\forall p\in E\).

Dense¶

Let \(X\) be a metric space, \(E\subset X\).

\(E\) is dense in \(X\) if every point of \(X\) is a limit point of \(E\), or a point of \(E\) (or both).

Closure¶

Let \(X\) be a metric space, \(E\subset X\).

Denote the set of all limit point s of \(E\subset X\), then the closure of \(E\) is the set \(\bar{E} = E\cup E'\).

Some theorems:

  • \(\bar{E}\) is a closed set.
  • \(E= \bar{E}\) if and only if \(E\) is a closed set.
  • \(\bar{E} \subset F\) for every closed set \(F\subset X\) such that \(E \subset F\).

De Morgan’s laws¶

Let \(\{E_{\alpha}\}\) be a collection of sets \(E_{\alpha}\), then

\[\begin{split}\left(\cup_{\alpha} E_{\alpha}\right)^c = \cap_{\alpha}(E_{\alpha}^c), \\ \left(\cap_{\alpha} E_{\alpha}\right)^c = \cup_{\alpha}(E_{\alpha}^c).\end{split}\]

Compact set¶

Open cover¶

Let \(X\) be a metric space, \(E\subset X\).

An open cover of \(E\subset X\) is the collection \(\{G_{\alpha}\}\) of open subsets of \(X\) such that \(E\subset \cup_{\alpha} G_{\alpha}\).

Compact¶

Let \(X\) be a metric space, \(K\subset X\).

\(K\) is said to be compact if every open cover of \(K\) contains a finite subcover.

In other words, \(\exists \alpha_1, \dots, \alpha_n\) (finitely many indices) such that

\[K \subset G_{\alpha_1} \cdots \cup G_{\alpha_n}\]

Some theorems:

  • Suppose \(K\subset Y\subset X\), then \(K\) is compact relative to \(X\) if and only if \(K\) is compact relative to \(Y\).
  • Compact subsets of metric spaces are closed.
  • Compact subsets of compact sets are compact.
  • If \(F\) is closed and \(K\) is compact, then \(F\cap K\) is compact.

Heine-Borel theorem¶

If a set \(E\in \mathbb{R}^k\) has one of the following three properties, then it has the other two:

  • \(E\) is closed and bounded.
  • \(E\) is compact.
  • Every infinite subset of \(E\) has a limit point in \(E\).
